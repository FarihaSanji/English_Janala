# 📘 English Janala – Interactive English Learning Platform

An interactive and responsive web application designed to help users learn English vocabulary through structured lessons and dynamic UI interaction.

---

## 🌐 Live Demo

🔗 https://farihasanji.github.io/English_Janala/

---

## 🎯 Project Overview

English Janala is a frontend-based learning platform where users can explore vocabulary through lesson-based navigation. The project focuses on building a real-world interactive UI using **vanilla JavaScript**, without any frameworks.

---

## ✨ Features

- 📚 Lesson-based vocabulary learning system  
- ⚡ Dynamic content rendering without page reload  
- 🧠 Interactive modal for word details  
- 🎨 Responsive UI using Tailwind CSS & DaisyUI  
- 📱 Mobile-friendly design  
- 🔄 Loading spinner for better user experience  

---

## 🧠 Concepts Used

- DOM Manipulation

- Responsive Web Design
